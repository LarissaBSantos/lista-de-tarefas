# Lista de Tarefas (Laravel)

Aplicação web simples para gerenciamento de tarefas pessoais, permitindo criar, listar, editar, marcar como concluídas e excluir tarefas.

## Descrição do Projeto
Este projeto será desenvolvido com o objetivo de praticar os conceitos fundamentais do framework Laravel, incluindo autenticação manual de usuários, uso do Blade, rotas, controllers e Eloquent ORM.

## Funcionalidades
- Cadastro e login de usuários (autenticação manual)
- Criação de tarefas
- Listagem de tarefas por usuário
- Edição e exclusão de tarefas
- Marcar tarefas como concluídas

## Tecnologias Utilizadas
- HTML, CSS e JavaScript
- SQLite
- Laravel 12
